# ForecastPilot AI — Frontend

This is the Next.js 15 frontend application for ForecastPilot AI.

For full documentation, architecture details, and setup instructions for both the frontend and backend, please see the **[Main Project README](../README.md)** located in the root directory.

## Local Development Quickstart

First, make sure the Python backend is running. Then, run the Next.js development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
